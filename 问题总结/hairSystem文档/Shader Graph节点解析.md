Shader Graph节点解析

一.输入

 **Bitangent Vector **— 副切线向量

访问mesh顶点或片元的副切线矢量 (Bitangent Vector)，具体取决于节点所属图形部分的有效着色器阶段(Shader Stage)。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                            |
| ------ | -------- | -------- | ---- | ------------------------------- |
| out    | OutPut   | Vector 3 | 无   | mesh顶点/片元的**副切线向量**。 |

**Instance ID** — 实例ID

Unity 使用 GPU 实例化渲染时，会为每个几何体分配一个实例 ID。使用此节点可在 Graphics.DrawMeshInstanced API 调用中捕获 Instance ID 值。当 Unity 不使用 GPU 实例化渲染时，此 ID 为 0。当 Unity 使用动态实例化时，多个帧中的实例 ID 可能不一致。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                                        |
| ------ | -------- | -------- | ---- | ------------------------------------------- |
| out    | OutPut   | float    | 无   | 通过实例化draw call生成的mesh的**实例id**。 |

**Normal Vector** — 法线向量

访问网格顶点或片元的法线向量。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                          |
| ------ | -------- | -------- | ---- | ----------------------------- |
| out    | OutPut   | Vector 3 | 无   | mesh顶点/片元的**法线向量**。 |

**Position** — 位置

访问网格顶点或片元的位置信息，具体取决于节点所属图形部分的有效着色器阶段(Shader Stage)。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                      |
| ------ | -------- | -------- | ---- | ------------------------- |
| out    | OutPut   | Vector 3 | 无   | mesh顶点/片元的**位置**。 |

**Screen Position** — 位置

访问网格顶点或片元的屏幕位置。可使用 Mode 下拉选单参数选择输出值的模式。

Default
返回屏幕位置。此模式将屏幕位置除以裁剪空间位置 W 分量。

Raw
返回屏幕位置。此模式不会将屏幕位置除以裁剪空间位置 W 分量。这对投影很有用。

Center
返回屏幕位置偏移，因此位置 float2(0,0) 位于屏幕的中心。

Tiled
返回屏幕位置偏移，因此位置 float2(0,0) 位于屏幕的中心，并使用 frac 平铺。

Pixel
以屏幕的实际像素宽度和高度值返回屏幕位置。在此模式下，位置 float2(0,0) 位于屏幕左下角。默认模式的范围始终为 0 至 1，而像素模式的范围取决于屏幕分辨率。该模式下不使用 Z 值和 W 值，因此它们始终为 0。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                 |
| ------ | -------- | -------- | ---- | -------------------- |
| out    | OutPut   | Vector 4 | 无   | mesh的**屏幕位置**。 |

**Tangent Vector** — 切线向量

访问网格顶点或片元的切线矢量。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                          |
| ------ | -------- | -------- | ---- | ----------------------------- |
| out    | OutPut   | Vector 3 | 无   | mesh顶点/片元的**切线向量**。 |

**UV** — UV坐标

访问mesh顶点或片元的 **UV** 坐标。可使用 **Channel** 下拉选单参数选择输出值的坐标通道。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                        |
| ------ | -------- | -------- | ---- | --------------------------- |
| out    | OutPut   | Vector 4 | 无   | mesh顶点/片元的**UV坐标**。 |

**Vertex Color** — 顶点颜色

访问mesh顶点或片元的顶点颜色值。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                            |
| ------ | -------- | -------- | ---- | ------------------------------- |
| out    | OutPut   | Vector 4 | 无   | mesh顶点/片元的**顶点颜色值**。 |

**Vertex ID** — 顶点ID

访问mesh顶点或片元的顶点ID值。

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                          |
| ------ | -------- | -------- | ---- | ----------------------------- |
| out    | OutPut   | Float    | 无   | mesh顶点/片元的**顶点ID值**。 |

**View Direction** — 视图方向向量

用于访问mesh顶点或片段的视图方向向量。这是从顶点或片段到摄像机的向量。选择坐标空间（Space）选项可修改输出值的坐标空间。

（在 11.0 版本之前，"视图方向节点 "在 HDRP 中的工作方式与 URP 中不同。在 URP 中，它只存储归一化的对象空间矢量。而 HDRP 则存储所有归一化向量。从 11.0 版开始，该节点将在高清渲染管道和通用渲染管道中存储所有归一化向量。）

| 针脚名 | 针脚类型 | 数据类型 | 绑定 | 描述                              |
| ------ | -------- | -------- | ---- | --------------------------------- |
| out    | OutPut   | Vector 3 | 无   | mesh顶点/片元的**视图方向向量**。 |
